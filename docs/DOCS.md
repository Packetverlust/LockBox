# LockBox — Docs

LockBox is an encrypted local password manager for the command line (and TUI).

## Topics

| Topic    | What it covers                              |
|----------|---------------------------------------------|
| overview | What LockBox is and how it works            |
| commands | All available commands and flags            |
| usage    | Common everyday usage patterns              |
| advanced | Power-user flags, multi-account setup       |
| security | Encryption, key derivation, what's on disk  |

```
lockbox docs overview
lockbox docs commands
lockbox docs usage
lockbox docs advanced
lockbox docs security
```
